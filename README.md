# PDF TOTATE TOOL项目介绍

一个基于 Next.js 和 React-pdf 的在线 PDF 旋转工具，完全在浏览器中运行，无需上传文件到服务器，保护您的隐私。

## 主要功能

- 创建、编辑和删除 Webhook
- 启用/禁用 Webhook 状态
- 自定义数据触发 Webhook
- 亮色/暗色主题支持
- 响应式三列布局

## 安装指南

```bash
# 克隆仓库
git clone https://github.com/keiSenFront/webhook-management.git
cd pdf-rotate-tool

# 安装依赖
yarn install

# 启动开发服务器
yarn dev
```

## 生产环境构建

```bash
yarn build
yarn start
```
